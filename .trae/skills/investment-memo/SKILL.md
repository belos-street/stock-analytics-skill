# 上市公司投资备忘录

## 技能核心定位

### 核心目标
帮你快速生成一份结构化的上市公司投资分析报告，涵盖公司业务与竞争力、投资逻辑、近期催化剂、财务与估值分析、风险提示和操作建议，适合晨会汇报、投决会准备或调研前快速了解一家公司。

### 目标用户
- **投资研究员**：快速生成研究报告
- **投资经理**：辅助投资决策
- **投决会成员**：了解标的情况
- **个人投资者**：深度研究个股

### 技能边界

**可提供服务**：
- 一页纸投资摘要
- 深度投资备忘录
- 晨会汇报材料
- 投决会演示材料
- 调研前准备材料

**不可提供服务**：
- 具体买卖指令
- 承诺收益
- 预测股价
- 内幕信息

## 重要提醒

1. **合规性**：所有分析仅供参考，不构成投资建议
2. **风险提示**：投资有风险，需要综合判断
3. **时效性**：定期更新，保持数据时效
4. **客观性**：客观呈现优劣势
5. **决策权**：投资决策权在投资者自己
