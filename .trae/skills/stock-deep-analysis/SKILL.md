# 个股深度分析Skill

## 技能核心定位

### 核心目标
为用户提供个股的专业深度分析，帮助用户理解个股的基本面、估值水平、技术走势和潜在风险，辅助用户做出更理性的投资决策。

### 目标用户
- **个股投资者**：投资个股的投资者
- **基本面投资者**：注重公司基本面的投资者
- **价值投资者**：寻找被低估股票的投资者
- **成长投资者**：寻找高成长股票的投资者

### 技能边界

**可提供服务**：
- 基本面分析：财务指标、业务模式、竞争优势
- 估值分析：PE、PB、PEG、DCF等估值方法
- 技术面辅助：趋势、支撑压力位、均线走势
- 风险评估：行业风险、公司风险、市场风险
- 投资逻辑梳理：投资逻辑、投资风险、注意事项

**不可提供服务**：
- 具体买卖指令
- 承诺收益
- 预测涨跌
- 内幕信息

### 财务指标分析

#### 1. 盈利能力指标

**毛利率**：
- 公式：毛利率 = (营业收入 - 营业成本) / 营业收入
- 解读：毛利率越高，说明公司产品竞争力越强
- 行业对比：与行业平均毛利率对比

**净利率**：
- 公式：净利率 = 净利润 / 营业收入
- 解读：净利率越高，说明公司盈利能力越强
- 趋势分析：近5年净利率变化

**ROE（净资产收益率）**：
- 公式：ROE = 净利润 / 股东权益
- 解读：ROE越高，说明股东投资回报越高
- 行业对比：与行业平均ROE对比

**ROA（总资产收益率）**：
- 公式：ROA = 净利润 / 总资产
- 解读：ROA越高，说明公司资产利用效率越高
- 趋势分析：近5年ROA变化

#### 2. 成长能力指标

**营收增长率**：
- 公式：营收增长率 = (本期营收 - 上期营收) / 上期营收
- 解读：营收增长越快，说明公司发展越好
- 趋势分析：近5年营收增长率变化

**净利润增长率**：
- 公式：净利润增长率 = (本期净利润 - 上期净利润) / 上期净利润
- 解读：净利润增长越快，说明公司盈利越好
- 趋势分析：近5年净利润增长率变化

**扣非净利润增长率**：
- 公式：扣非净利润增长率 = (本期扣非净利润 - 上期扣非净利润) / 上期扣非净利润
- 解读：扣除非经常性损益后的净利润，更反映公司真实盈利能ason

#### 3. 财务健康指标

**资产负债率**：
- 公式：资产负债率 = 总负债 / 总资产
- 解读：资产负债率越低，财务风险越小
- 行业对比：与行业平均资产负债率对比

**流动比率**：
- 公式：流动比率 = 流动资产 / 流动负债
- 解读：流动比率越高，短期偿债能力越强
- 合理范围：1.5-2.0

**速动比率**：
- 公式：速动比率 = (流动资产 - 存货) / 流动负债
- 解读：速动比率越高，短期偿债能力越强
- 合理范围：1.0-1.5

#### 4. 现金流指标

**经营现金流/营收**：
- 公式：经营现金流/营收 = 经营现金流净额 / 营业收入
- 解读：比值越高，说明公司收入质量越好
- 合理范围：10%以上

**自由现金流**：
- 公式：自由现金流 = 经营现金流 - 资本支出
- 解读：自由现金流越多，说明公司越能创造价值
- 趋势分析：近5年自由现金流变化

**分红率**：
- 公式：分红率 = 股息支付 / 净利润
- 解读：分红率越高，说明公司回报股东越多
- 行业对比：与行业平均分红率对比

## 估值分析

### 相对估值法

#### 1. 市盈率（PE）

**基本公式**：
- PE = 股票价格 / 每股收益
- PE = 总市值 / 净利润

**类型**：
- 静态PE：使用上一年度净利润
- 动态PE：使用预期本年度净利润
- TTM PE：使用近12个月净利润

**估值判断**：
- PE < 10：低估
- PE 10-20：合理
- PE 20-30：高估
- PE > 30：泡沫

**适用场景**：
- 成熟行业：适用性强
- 稳定盈利：适用性强
- 周期行业：需结合周期阶段
- 亏损公司：不适用

**使用方法**：
- 历史分位数：对比历史PE
- 行业对比：与行业平均PE对比
- 成长性调整：高成长给予更高PE

#### 2. 市净率（PB）

**基本公式**：
- PB = 股票价格 / 每股净资产
- PB = 总市值 / 净资产

**估值判断**：
- PB < 1：破净，低估
- PB 1-2：合理
- PB 2-3：高估
- PB > 3：泡沫

**适用场景**：
- 金融行业：银行、保险、券商
- 制造业：重资产行业
- 轻资产行业：参考性较弱

**使用方法**：
- 历史分位数：对比历史PB
- 行业对比：与行业平均PB对比
- ROE调整：高ROE给予更高PB

#### 3. 市销率（PS）

**基本公式**：
- PS = 股票价格 / 每股营收
- PS = 总市值 / 营业收入

**估值判断**：
- PS < 1：低估
- PS 1-3：合理
- PS 3-5：高估
- PS > 5：泡沫

**适用场景**：
- 成长型公司：尚未盈利
- 互联网公司：用户价值
- 零售行业：门店价值

#### 4. 股息率

**基本公式**：
- 股息率 = 每股股息 / 股票价格

**估值判断**：
- 股息率 > 4%：高股息
- 股息率 2-4%：中等股息
- 股息率 < 2%：低股息

**适用场景**：
- 成熟行业：银行、公用事业
- 高分红公司：蓝筹股

### 绝对估值法

#### 1. DCF（现金流折现）

**基本原理**：
- 将公司未来现金流折现到当前
- 折现率 = 无风险利率 + 风险溢价

**步骤**：
1. 预测未来现金流
2. 确定折现率
3. 计算现值
4. 得出公司价值

**优点**：
- 考虑公司长期价值
- 不受市场情绪影响

**缺点**：
- 预测难度大
- 参数敏感度高

#### 2. 分部估值

**方法**：
- 将公司不同业务分别估值
- 汇总得出公司总价值

**适用场景**：
- 多元化公司
- 业务差异大

### 估值分析流程

#### 第一步：选择估值方法

**根据行业选择**：
- 成熟行业：PE为主
- 金融行业：PB为主
- 成长行业：PS为主
- 分红公司：股息率为主

#### 第二步：收集数据

**财务数据**：
- 营收、净利润
- 净资产
- 现金流
- 分红

**市场数据**：
- 股价
- 市值
- 成交量

#### 第三步：计算估值

**计算各项指标**：
- PE、PB、PS、股息率
- 历史分位数
- 行业对比

#### 第四步：得出结论

**估值判断**：
- 综合各项指标判断
- 考虑成长性
- 考虑风险因素

## 技术面分析

### 趋势分析

#### 1. 长期趋势

**判断方法**：
- 股价是否处于上升通道
- 均线是否多头排列
- 高点是否逐步抬高

**趋势信号**：
- 上升趋势：高点抬高、低点抬高
- 下降趋势：高点降低、低点降低
- 横盘趋势：区间震荡

#### 2. 中期趋势

**判断方法**：
- 60日均线方向
- 120日均线方向
- 中期趋势线

#### 3. 短期趋势

**判断方法**：
- 5日均线方向
- 10日均线方向
- 短期趋势线

### 均线分析

#### 均线系统

**常用均线**：
- 5日均线：短期均线
- 10日均线：短期均线
- 20日均线：中期均线
- 60日均线：中期均线
- 120日均线：长期均线
- 250日均线：年线

**均线形态**：
- 多头排列：短期均线在长期均线上方
- 空头排列：短期均线在长期均线下方
- 金叉：短期均线上穿长期均线
- 死叉：短期均线下穿长期均线

**支撑强度**：
- 次数越多，支撑越强

#### 压力位

**压力位类型**：

**压力强度**：

## 风险评估

### 行业风险

#### 1. 行业周期风险

**周期类型**：
- 强周期行业：钢铁、有色、化工
- 弱周期行业：消费、医药、公用事业

**风险表现**：
- 行业下行期，业绩大幅下降
- 股价波动较大

**应对方法**：
- 关注行业景气度
- 避免在行业高峰买入
- 考虑行业轮动

#### 2. 行业政策风险

**政策类型**：
- 产业政策
- 环保政策
- 监管政策

**风险表现**：
- 政策变化影响行业发展
- 政策利空导致股价下跌

**应对方法**：
- 关注政策变化
- 分散行业配置
- 避免政策敏感行业

#### 3. 行业竞争风险

**竞争类型**：
- 行业内竞争
- 跨行业竞争
- 国际竞争

**风险表现**：

**应对方法**：
- 关注公司竞争力

## 常见问题解答

### Q1：如何判断一只股票的好坏？

**A**：判断股票好坏主要看：
1. 基本面：盈利能力、成长能力、财务健康
2. 估值：PE、PB是否合理
3. 行业：行业前景和竞争格局
4. 管理：管理层能力和激励机制

### Q2：股票估值多少合理？

**A**：估值合理性判断：
- 成熟行业：PE 10-20倍合理
- 成长行业：PE 20-40倍合理
- 周期行业：PE需要结合周期
- 具体公司需要具体分析

### Q3：股票为什么会下跌？

**A**：下跌原因：
1. 基本面恶化
2. 估值过高
3. 市场下跌
4. 政策利空
5. 资金流出

### Q4：股票长期不涨怎么办？

**A**：处理方法：
1. 分析原因：基本面问题还是估值问题
2. 重新评估：是否值得继续持有
3. 设定目标：设定卖出价位
4. 分散投资：不要集中单一股票

### Q5：股票分红有用吗？

**A**：分红的作用：
1. 现金流回报
2. 复利效应（红利再投资）
3. 反映公司盈利能力

### Q6：如何控制股票投资风险？

**A**：风险控制方法：
1. 分散投资：不要集中单一股票
2. 控制仓位：不要满仓操作
3. 设置止损：控制最大亏损
4. 持续跟踪：关注基本面变化
