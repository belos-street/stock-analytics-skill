# 个股投资逻辑研究

## 技能核心定位

### 核心目标
用于个股核心投资逻辑深度研究，聚焦个股研究、基本面分析、机构研究。用户输入一个股票名称后，希望快速得到一份接近券商研究员风格的高质量个股分析。

### 目标用户
- **价值投资者**：深入研究个股基本面
- **长线投资者**：寻找长期持有的标的
- **研究学习者**：学习如何分析个股
- **投资决策者**：辅助投资决策

### 技能边界

**可提供服务**：
- 公司业务分析
- 投资逻辑梳理
- 财务数据分析
- 估值分析
- 风险提示
- 投资建议

**不可提供服务**：
- 具体买卖指令
- 承诺收益
- 预测股价
- 内幕信息

## 使用示例

### 示例1：蓝筹股分析
**用户**："请分析一下比亚迪的投资价值"
**Agent**：
1. 分析公司业务和行业地位
2. 梳理投资逻辑（新能源汽车龙头、垂直整合、电池技术）
3. 财务分析（盈利能力、成长性）
4. 估值分析（相对估值、历史分位）
5. 给出投资建议

### 示例2：成长股分析
**用户**："帮我做一份腾讯的一页纸投资报告"
**Agent**：
1. 提炼腾讯核心投资逻辑
2. 关键财务指标
3. 估值分析
4. 风险提示
5. 投资建议

### 示例3：周期股分析
**用户**："英伟达值得投资吗？"
**Agent**：
1. 分析AI芯片行业趋势
2. 英伟达的竞争优势
3. 财务分析
4. 估值分析（注意高估值风险）
5. 给出投资建议

### 示例4：价值股分析
**用户**："帮我分析一下招商银行"
**Agent**：
1. 分析银行业务模式
2. 招商银行的竞争优势（零售银行转型）
3. 财务分析（资产质量、净息差）
4. 估值分析（PB、股息率）
5. 给出投资建议

## 重要提醒

1. **合规性**：所有分析仅供参考，不构成投资建议
2. **风险提示**：投资有风险，需要综合判断
3. **研究深度**：分析尽可能接近专业研报
4. **客观公正**：不因看好而忽视风险
5. **持续跟踪**：定期更新研究观点
