- **投资者类型**：普通新手，无盯盘时间

## 持仓分析报告流程

当用户要求分析持仓时，按照以下流程：

### 第一步：读取持仓配置
- 读取 `position.md` 文件，了解投资者的持仓配置
- 了解各持仓标的的类型、代码、目标占比

### 第二步：整体市场环境分析
调用 `stock-analytics` 了解：
- 今日A股整体表现（指数涨跌、成交量、市场情绪）
- 资金流向（北向资金、南向资金）
- 热点板块

### 第三步：分项持仓分析
根据持仓配置，调用对应的专业Skill进行分析

### 第四步：资产配置评估
调用 `asset-allocation-rebalancing` 评估：
- 当前配置状态
- 是否需要再平衡
- 后续操作建议：
  - **加仓建议**：哪些标的处于低估区间、具备加仓条件
  - **减仓建议**：哪些标的处于高估区间、考虑减仓
  - **定投建议**：哪些标的适合定投、定投频率和金额
  - **持有建议**：哪些标的继续持有不动
  - **换仓建议**：是否有更好的替代标的

### 第五步：总结输出
综合以上分析，给出：
- 今日整体表现总结
- 各持仓标的分析
- 操作建议
- 风险提示

## 今日持仓日报生成流程

当用户要求生成"今日持仓分析报告"时，按照以下流程执行：

### 第一步：获取今日市场行情数据

**调用工具**：WebSearch 搜索当日市场数据

**搜索内容**：
- "今日A股市场行情 收盘"
- "北向资金 南向资金流向"
- "红利低波ETF 515450 收盘"
- "A500ETF 563360 收盘"
- "自由现金流ETF 159201 收盘"
- "中国移动 600941 A股 收盘"
- "招商银行 600036 收盘"
- "恒生医疗ETF 513060 收盘"
- "恒生科技ETF 513010 收盘"

**关键数据**：
- 三大指数涨跌幅（上证指数、深证成指、创业板指）
- 成交量和成交额
- 资金流向（北向/南向资金）
- 市场情绪和热点板块

### 第二步：汇总各持仓标的今日表现

**必须覆盖的持仓标的**（共9个）：

| 标的类型 | 代码 | 优先级 |
|---------|------|-------|
| 红利低波50ETF | 515450 | 必须 |
| 自由现金流ETF | 159201 | 必须 |
| A500ETF | 563360 | 必须 |
| 中国移动（A股） | 600941 | 必须 |
| 招商银行 | 600036 | 必须 |
| 恒生医疗ETF | 513060 | 必须 |
| 恒生科技ETF | 513010 | 必须 |
| 债券基金 | 006493/006961/002276/217024 | 必须 |
| 现金/货币基金 | - | 必须 |

### 第三步：生成持仓日报模板

**文件命名**：`reports/daily_report_YYYYMMDD.md`

**报告模板结构**：

```markdown
# 持仓日报 — YYYY年MM月DD日

## 今日市场概况
- 三大指数涨跌幅表格
- 市场特征描述
- 资金流向

## 持仓标的今日表现
- 标的涨跌表格（必须包含全部9个持仓）
- 整体评估

## 今日操作建议
- 建议持有不动表格
- 核心观点

## 风险提示
- 合规声明
- 风险提示
```

### 第四步：输出报告

1. 将报告保存至 `reports/` 目录
2. 在对话中向用户展示报告摘要
3. 提示用户报告已保存

2. **风险提示**：市场有风险，投资需谨慎
3. **长期视角**：坚持长期持有、分红再投入、不波段、不追热点、年度再平衡
4. **个性化**：根据个人情况做出投资决策
