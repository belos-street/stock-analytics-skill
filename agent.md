## 数据获取工具

### 本地 CLI 工具

项目提供本地命令行工具，可直接获取股票/基金实时数据：

**使用方法**：
```bash
# 进入项目目录
cd {{PROJECT_PATH}}

# 运行 CLI
bun run main.ts [options]
```

**可用参数**：

| 参数             | 说明               | 示例                    |
| -------------- | ---------------- | --------------------- |
| `-s, --stocks` | 股票代码（逗号分隔）       | `-s hk00700,sh000001` |
| `-f, --funds`  | 基金代码（逗号分隔）       | `-f 320007,110022`    |
| `-o, --format` | 输出格式: raw \| llm | `-o llm`              |
| `-h, --help`   | 查看帮助             | <br />                |

**输出格式**：
- `llm`（默认）：大模型友好的统一格式，包含 type 字段
- `raw`：原始数据结构

**示例命令**：
```bash
# 查询腾讯控股股票
cd {{PROJECT_PATH}} && bun run main.ts -s hk00700

# 查询多只股票
cd {{PROJECT_PATH}} && bun run main.ts -s hk00700,hk01810

# 查询基金
cd {{PROJECT_PATH}} && bun run main.ts -f 320007

# 同时查询股票和基金
cd {{PROJECT_PATH}} && bun run main.ts -s hk00700 -f 320007
```

**支持的市场**：
- 港股：`hk00700`（腾讯控股）、`hk01810`（小米集团）等
- A股：`sh000001`（上证指数）、`sh600519`（贵州茅台）等
- 基金：天天基金网支持的基金代码

## 持仓分析报告流程

当用户要求分析持仓时，按照以下流程：

### 第一步：读取持仓配置
- 读取 `{{POSITION_FILE}}` 文件，了解投资者的持仓配置
- 了解各持仓标的的类型、代码、目标占比

### 第二步：获取实时数据
优先使用本地 CLI 工具获取股票/基金实时数据：

```bash
cd {{PROJECT_PATH}}
bun run main.ts -s hk00700,hk01810
bun run main.ts -s sh600519,sh000001
```

### 第三步：整体市场环境分析
调用 `stock-analytics` 了解：
- 今日A股整体表现（指数涨跌、成交量、市场情绪）
- 资金流向（北向资金、南向资金）
- 热点板块

### 第四步：分项持仓分析
根据持仓配置，调用对应的专业Skill进行分析

### 第五步：资产配置评估
调用 `asset-allocation-rebalancing` 评估：
- 当前配置状态
- 是否需要再平衡
- 后续操作建议：
  - **加仓建议**：哪些标的处于低估区间、具备加仓条件
  - **减仓建议**：哪些标的处于高估区间、考虑减仓
  - **定投建议**：哪些标的适合定投、定投频率和金额
  - **持有建议**：哪些标的继续持有不动
  - **换仓建议**：是否有更好的替代标的

### 第六步：总结输出
综合以上分析，给出：
- 今日整体表现总结
- 各持仓标的分析
- 操作建议
- 风险提示

当用户要求生成"今日持仓分析报告"时，按照以下流程执行：

```bash

# 查询基金持仓
bun run main.ts -f 320007,110022 -o llm

# 查询A股持仓
bun run main.ts -s sh600519,sh000001 -o llm
```

**CLI 输出格式**（大模型友好）：
```json
[
  {
    "type": "stock",
    "code": "TENCENT",
    "name": "腾讯控股",
    "current_price": 538,
    "change": 22,
    "change_percent": 4.264,
    "update_time": "2026/03/10 11:09"

]
```

**备选方案**：如 CLI 无法获取某些标的，使用 WebSearch 补充搜索

- `fundcode`：基金代码

- `jzrq`：净值日期
