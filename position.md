# 个人长期复利投资策略

## 查询命令
```bash
# 基金（债基 + QDII ETF）
bun main.ts -f "006493,006961,002276,217024,159201"

# 股票/A股ETF
bun main.ts -s "sh600941,sh600036,sh515450,sh563360,sh513060,sh513010"
```

## 一、核心定位
+ 投资者：普通新手，无盯盘时间，追求**长期稳健复利**，做时间的朋友
+ 风险偏好：中低风险为主，小仓位高弹性进攻，严控回撤
+ 总资产：100万元
+ 核心原则：长期持有、分红再投入、不波段、不追热点、年度再平衡

## 查询参数说明

| 类型 | 查询参数 | 代码格式 | 示例 |
| --- | --- | --- | --- |
| 基金（债基、QDII ETF） | -f | 无前缀 | `006493`, `159201` |
| A股/ETF | -s | sh + 代码（不带.） | `sh600941`, `sh515450` |
| 港股 | -s | hk + 代码（不带.） | `hk0941` |

## 代码前缀说明

- **sh.** - 上海证券交易所（A股）
- **sz.** - 深圳证券交易所（A股）
- **hk.** - 香港证券交易所（港股）
- **无前缀** - 基金代码（天天基金网）
